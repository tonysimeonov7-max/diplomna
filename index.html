1. Увод

История на операционните системи

1960s

Първите операционни системи

IBM OS/360 и MULTICS поставят основите на модерните ОС с пакетна обработка на задачи.

1970s

Раждането на UNIX

UNIX става революционна система с команден интерфейс, влияеща върху всички съвременни ОС.

1980s

Графичният интерфейс се ражда

Apple Macintosh и Microsoft Windows правят компютрите достъпни за масовия потребител.

2020s

Хибридни системи и AI

Съвременните ОС комбинират CLI и GUI, интегрират изкуствен интелект и облачни технологии.

Видове операционни системи

Еднопотребителски

MS-DOS, ранни версии на Windows

Многопотребителски

UNIX, Linux, Windows Server

Еднозадачни

MS-DOS, ранни мобилни ОС

Многозадачни

Windows, Linux, macOS

2. Характеристики на CLI и GUI

Команден интерфейс (CLI)
  • Текстово базиран интерфейс
  • Високопроизводителен
  • Минимална консумация на ресурси
  • Автоматизация чрез скриптове
  • Прецизен контрол
  • Дистанционно управление
Пример: $ sudo apt update && sudo apt upgrade
Графичен интерфейс (GUI)
  • Визуално базиран интерфейс
  • Интуитивен за използване
  • Подходящ за начинаещи
  • Визуализация на данни
  • Drag & Drop функционалност
  • Богата визуална обратна връзка
Пример:

Start → Settings → Update & Security

Общи компоненти на операционните системи

Ядро (Kernel)

Управлява хардуерните ресурси

Управление на паметта

Разпределяне и контрол на RAM

Файлова система

Организация и съхранение на данни

Драйвери

Комуникация с хардуера

Сигурност

Защита на системата и данните

Потребителски интерфейс

4. Архитектура на операционните системи

Монолитно ядро

Единно голямо ядро

Всички услуги в kernel space

  • Висока производителност
  • По-малка сигурност
  • Трудна поддръжка

Примери: Linux, UNIX, MS-DOS

Микроядро

Услуги в user space
Минимално ядро
  • По-висока сигурност
  • Лесна поддръжка
  • Услуги в потребителско пространство

Примери: Minix, QNX, Mach

7. Заключение

Обобщение на резултатите

Сравнителният анализ показва, че и двата подхода имат уникални предимства:

  • CLI системите са оптимални за сървъри, автоматизация и DevOps.
  • GUI системите са незаменими за офис приложения, дизайн и мултимедия.
  • Хибридните решения предлагат най-доброто от двата свята.
